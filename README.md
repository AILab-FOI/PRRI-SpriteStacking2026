# PRRI-SpriteStacking2026

A sprite stacking based cozy RPG game developed by students and the [Artificial Intelligence Laboratory](https://ai.foi.hr/) at the [University of Zagreb Faculty of Organization and Informatics](https://www.foi.unizg.hr/). The game is developed using Python and GamePy. More details available at [itch.io](https://ailab-foi.itch.io/prri-spritestacking2026). 

# Credits

This game is based on [Stanislav Petrov's implementation](https://github.com/StanislavPetrovV/SpriteStacking) licensed under the MIT license. A previous version of this game has been developed [here](https://github.com/AILab-FOI/PRRI-ST2025).

Splash screen graphics have been generated using AI (DALL-E) and edited.
